📡 Connect Google Apps Script

Enter your deployed Google Apps Script URL to connect your portfolio to Google Sheets.

🚀 First Time Setup Guide

Follow these steps to connect your Google Sheet:

  1. Open Google Sheets — Go to sheets.new to create a new spreadsheet. Name it "My Portfolio".
  2. Open Apps Script — In your spreadsheet, click Extensions → Apps Script.
  3. Paste the Code — Delete any existing code, then paste the entire contents of Code.gs from this project.
  4. Initialize Data — In Apps Script editor, select the function initDefaultData from the dropdown and click ▶ Run. Allow permissions when asked.
  5. Deploy as Web App — Click Deploy → New Deployment. Type: Web app. Execute as: Me. Who has access: Anyone. Click Deploy and copy the URL.
  6. Paste URL above — Paste the deployment URL in the field above and click Save.
  7. Login Again — Reload this page and login with password portfolio123. Then change your password!
⚠️ Important: After editing your Google Apps Script code, you must create a new deployment each time (not redeploy existing) to get an updated URL with new changes.
